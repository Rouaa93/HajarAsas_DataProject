# حجر أساس | Hajar Asas Data Project

This project aims to develop a data analytics and visualization system for a local concrete block factory "Hajar Asas", combining Python data analysis and a simple company website prototype.

## Project Goals
- Collect and analyze sales and production data.
- Build visual dashboards for insights.
- Support data-driven decision-making for production and marketing.
- Create a simple website for company visibility.

## Tools
- Python (pandas, matplotlib, seaborn, sklearn)
- HTML / React for front-end prototype
- GitHub for version control
